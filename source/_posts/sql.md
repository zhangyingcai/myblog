# 重要点

* SQL 对大小写不敏感
* 建议 SQL 语句都带分号

SQL 分为两个部分 数据操作语言(DML) 和 数据定义语言(DDL)

DML

* SELECT -获取数据
* UPDATE -更新数据
* DELETE -删除数据
* INSERTER INTO -插入数据

部分 DDL

* CREATE DATABASE - 创建新数据库
* ALTER DATABASE - 修改数据库
* CREATE TABLE - 创建新表
* ALTER TABLE - 变更（改变）数据库表
* DROP TABLE - 删除表
* CREATE INDEX - 创建索引（搜索键）
* DROP INDEX - 删除索引
